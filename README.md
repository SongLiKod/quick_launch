# Quick Launch

一款 Windows 桌面快速启动工具，支持应用程序、文件、文件夹、命令、链接等多种启动方式，提供全局快捷键、分组管理、系统托盘、主题切换等功能。

**版本：** 1.1.6

## 功能特性

### 🚀 启动项管理

| 类型 | 说明 | 示例 |
|------|------|------|
| **应用程序** | 启动 `.exe` 可执行文件 | `C:\Program Files\...\app.exe` |
| **批处理** | 运行 `.bat` / `.cmd` 脚本 | `C:\scripts\deploy.bat` |
| **文件** | 用关联程序打开文件 | `C:\docs\report.pdf` |
| **文件夹** | 在资源管理器中打开 | `D:\Projects` |
| **命令** | 在 CMD 窗口中执行命令 | `ipconfig /all`、`ping 8.8.8.8` |
| **链接** | 用默认浏览器打开链接 | `https://www.baidu.com` |
| **系统命令** | 内置系统操作（一键直达） | 关机、重启、锁屏、控制面板等 |

内置系统命令：关机、重启、锁屏、控制面板、回收站、此电脑、计算器、记事本、任务管理器。

### 📂 分组管理

- **创建/重命名/删除分组** — 自由管理启动项分类
- **分组颜色标识** — 每个分组可设置独立颜色
- **分组筛选** — 顶部筛选栏一键切换"全部分组"或指定分组
- **分组归属提示** — 在"全部"视图下显示每个启动项所属的分组名称标签
- **分组全局快捷键** — 为每个分组设置独立热键（如 `Ctrl+Alt+W`），按下后弹出分组选择面板
- **分组弹窗快速启动** — 分组快捷键弹窗中按 `A-Z` 字母键快速启动对应项

### ⌨️ 全局快捷键

- **启动项热键** — 为每个启动项设置自定义全局热键（如 `Ctrl+Alt+A`）
- **显示/隐藏窗口快捷键** — 如 `Ctrl+Shift+Q`，后台一键唤出
- **分组热键** — 按分组快速唤出该组启动项选择面板
- **快捷键冲突检测** — 设置时自动检测是否已被其他项目占用

### 📊 多种排序方式

| 排序模式 | 说明 |
|----------|------|
| **手动拖拽** | 长按拖拽手柄自由排序 |
| **按名称** | 按字母顺序排序 |
| **按创建时间** | 按添加时间排序 |
| **按类型** | 按启动项类型分组排列 |
| **最常用** | 按启动次数降序排列，次数相同时按最近使用时间 |
| **最近使用** | 按最近启动时间降序排列 |

### 📐 多列布局

支持 **1-4 列** 网格布局，在设置中调整列数，充分利用屏幕空间。

### 🔍 搜索过滤

顶部搜索框支持按名称和路径实时筛选启动项，快速定位目标。

### 🎨 主题模式

- **浅色模式** — 默认主题
- **深色模式** — 暗色界面
- **跟随系统** — 自动适配 Windows 主题

### 🖼️ 自定义图标

- 支持自定义窗口和系统托盘图标（`.ico` 文件）
- 可随时恢复为默认图标

### 📌 窗口置顶

开启后窗口始终显示在其他窗口之上，方便快速访问。

### 🗔 系统托盘

- 关闭时**最小化到托盘**，后台运行
- 托盘图标左键显示窗口，右键弹出快捷菜单
- **启动时隐藏到托盘** — 开机自启后默默在后台运行
- 开机自启支持**延迟启动**（0-60 秒），避免与其他开机程序冲突

### ⏱️ 启动统计与陈旧检测

- 自动记录每个启动项的**启动次数**和**最近启动时间**
- 超过 30 天未使用的启动项自动显示"30天未使用"标记

### 📋 启动日志

- 自动记录每次启动操作（成功/失败）
- 显示时间、名称、路径、状态、错误信息
- 上限 200 条，支持一键清空

### 🔄 配置导入导出

- **导出配置** — 将所有启动项和设置导出为 JSON 文件
- **导入配置** — 从 JSON 文件恢复/追加启动项（自动跳过重复项）

### ⚡ 管理员运行

支持为每个启动项单独设置**以管理员身份运行**。

### 🔔 版本更新检查

- 启动时自动检查 GitHub 最新 Release
- 发现新版本时弹窗提醒，一键跳转下载

## 安装

### 从 Release 下载

1. 前往 [Releases](https://github.com/qq1144403442/quick_launch/releases) 页面
2. 下载最新版本的 `quick_launch_windows.zip`
3. 解压到任意目录，运行 `quick_launch.exe`

### 自行构建

```bash
# 克隆仓库
git clone https://github.com/qq1144403442/quick_launch.git
cd quick_launch

# 安装依赖
flutter pub get

# 构建 Windows 版本
flutter build windows --release

# 产物在 build/windows/x64/runner/Release/
```

## 使用说明

### 添加启动项

1. 点击右下角 **`+`** 按钮
2. 选择类型：
   - **文件/文件夹** — 通过文件选择器选取
   - **命令** — 输入任意 CMD 命令
   - **链接** — 输入 URL 链接
3. 输入路径或命令，填写名称
4. 可选：设置快捷键、以管理员身份运行、选择所属分组
5. 点击 **添加**

### 管理分组

1. 点击分组栏右侧的编辑图标打开分组管理器
2. 点击 **添加分组** 创建新分组
3. 为分组设置名称、颜色、快捷键
4. 在添加/编辑启动项时可选择其所属分组

### 使用分组快捷键

1. 在分组管理器中为分组设置全局快捷键
2. 按下快捷键后弹出该分组的选择面板
3. 使用 `A-Z` 字母键快速启动项目，或方向键选择后按 `Enter`

### 设置全局快捷键

1. 在添加/编辑对话框中点击快捷键输入框
2. 输入格式如 `Ctrl+Alt+A`、`Win+E`、`Alt+F4`、`Shift+F5`
3. 如果快捷键被其他项目占用，会弹出冲突提示

### 设置显示窗口快捷键

进入设置 → 行为 → 显示窗口快捷键，设置全局热键用于在任意状态下唤出主窗口。

### 编辑/删除启动项

点击卡片上的 ▶️ 启动、✏️ 编辑或 🗑️ 删除按钮。

### 调整排序方式

进入设置 → 列表管理 → 排序方式，选择手动拖拽、按名称、按创建时间、按类型、最常用或最近使用。

### 配置导入导出

进入设置 → 列表管理，点击**导出配置**或**导入配置**进行数据备份和恢复。
